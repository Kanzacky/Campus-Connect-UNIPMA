Campus-Connect

Universitas PGRI Madiun

Platform Terpadu Manajemen Organisasi dan Kegiatan Kemahasiswaan.

Pemrograman Web 2

Latar Belakang Masalah

Informasi Tersebar

Pengumuman dan jadwal kegiatan Ormawa/UKM belum terpusat dalam satu wadah digital.

Sistem Konvensional

Pendaftaran keanggotaan mahasiswa masih dilakukan secara manual dan minim integrasi data.

Sulit Dimonitor

Pihak universitas (Admin) mengalami kesulitan melakukan pemantauan kegiatan secara real-time.

Solusi yang Ditawarkan

Single Page Application (SPA)

Platform web yang beroperasi mulus tanpa reload halaman untuk interaksi yang cepat.

Role-Based Access Control (RBAC)

Pemisahan Hak Akses yang tegas: Admin Kampus, Pengurus Organisasi, Anggota (Mahasiswa).

Manajemen Satu Pintu

Integrasi kelola kegiatan, pengumuman, dan data anggota secara terpusat.

Tech Stack

Backend Server

  • Laravel 13 & PHP 8.3
  • Autentikasi Laravel Sanctum
  • Arsitektur MVC

Frontend Client

  • React 19 & Inertia.js v3
  • Tailwind CSS v4
  • Komponen Radix UI
Menggunakan standar industri modern, sesuai spesifikasi UAS.
